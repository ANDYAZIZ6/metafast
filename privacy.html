Ringkasan

Ekstensi MetaFast for Adobe Stock membantu kontributor Adobe Stock membuat judul, kata kunci, dan kategori secara otomatis untuk mempercepat proses unggah. Ekstensi ini tidak mengumpulkan, menyimpan, menjual, atau membagikan data pribadi pengguna.

Izin & Penggunaan

Host Permissions

Akses ke https://*.adobe.com/ untuk membaca/menulis metadata di halaman unggah Adobe Stock. Akses dibatasi hanya pada domain yang relevan.

Tabs

Mendeteksi tab aktif saat pengguna berada di Adobe Stock. Data tidak disimpan atau dikirim ke pihak mana pun.

Side Panel

Menampilkan antarmuka MetaFast di panel samping untuk workflow yang lebih cepat dan efisien.

Storage

Menyimpan preferensi lokal seperti pengaturan strategi kata kunci. Semua data tetap di perangkat Anda.

Scripting

Menyuntikkan skrip untuk mengisi/membaca field metadata pada halaman Adobe Stock atas tindakan pengguna.

Catatan: Kami tidak menggunakan kode jarak jauh (remote code) atau eval(). Seluruh kode dibundel di paket ekstensi untuk keamanan maksimal.

Data yang Dikumpulkan

MetaFast tidak mengumpulkan informasi identitas pribadi, kesehatan, keuangan, autentikasi, komunikasi, lokasi, maupun histori web. Konten yang diakses terbatas pada field metadata di halaman Adobe Stock untuk menjalankan fungsi ekstensi.

Berbagi Data ke Pihak Ketiga

Kami tidak menjual atau mentransfer data pengguna ke pihak ketiga untuk tujuan apa pun. Privasi Anda adalah prioritas utama kami.

Keamanan & Anak-anak

Ekstensi ini tidak dirancang untuk anak di bawah 13 tahun dan tidak memproses data anak.

Summary

MetaFast for Adobe Stock helps Adobe Stock contributors automatically generate titles, keywords, and categories to speed up the upload process. This extension does not collect, store, sell, or share personal user data.

Permissions & Use

Host Permissions

Access to https://*.adobe.com/ to read/write metadata on Adobe Stock upload pages. Access is limited to relevant domains only.

Tabs

Detects active tab when user is on Adobe Stock. No data is stored or transmitted to any party.

Side Panel

Displays MetaFast interface in side panel for faster and more efficient workflow.

Storage

Stores local preferences like keyword strategy settings. All data remains on your device.

Scripting

Injects scripts to fill/read metadata fields on Adobe Stock pages upon user action.

Note: We do not use remote code or eval(). All code is bundled in the extension package for maximum security.

Data Collection

MetaFast does not collect personal identification information, health, financial, authentication, communication, location, or web history data. Content access is limited to metadata fields on Adobe Stock pages to perform extension functions.

Third Party Sharing

We do not sell or transfer user data to third parties for any purpose. Your privacy is our top priority.

Security & Children

This extension is not designed for children under 13 and does not process children's data.

This policy may be updated at any time. The "Last updated" date will be adjusted accordingly. If changes are material, we will update the listing description and this policy.

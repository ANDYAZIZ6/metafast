FAQ

Pertanyaan Umum

Apakah MetaFast mendukung berbagai format file?

Ya! MetaFast dirancang untuk menangani hampir semua aset digital Anda, termasuk format gambar (JPG, PNG), video (MP4, MOV), dan bahkan file vektor seperti Adobe Illustrator (.AI), .EPS, dan .SVG.

Bagaimana MetaFast menjaga keamanan file saya?

Keamanan adalah prioritas utama. MetaFast berjalan 100% lokal di komputer Anda. Hanya thumbnail gambar terkompresi dan beresolusi sangat rendah yang dikirim ke penyedia AI untuk dianalisis. File asli Anda tidak pernah meninggalkan komputer Anda.

Bisakah saya mengedit metadata sebelum ekspor?

Tentu! Ini adalah fitur andalan MetaFast versi Lengkap. Anda akan mendapatkan tab "Review & Edit" di mana Anda bisa meninjau dan menyempurnakan setiap judul, deskripsi, dan kata kunci sebelum file akhir disimpan.

Platform apa saja yang didukung?

Untuk saat ini, MetaFast hanya tersedia secara eksklusif untuk Windows 10 dan Windows 11 (64-bit). Dukungan untuk macOS dan Linux sedang dalam pengembangan.

Apakah ada garansi uang kembali?

Ya! Kami menawarkan garansi uang kembali 30 hari penuh. Jika Anda tidak puas dengan MetaFast, hubungi support kami dan dapatkan refund 100%.

Bagaimana cara kerja fitur AI Quality Control?

AI Quality Control kami menggunakan machine learning untuk memindai setiap gambar dan mendeteksi cacat anatomi, logo tersembunyi, atau potensi masalah IP lainnya secara otomatis dengan akurasi tinggi.

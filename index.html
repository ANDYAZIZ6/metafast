Frequently Asked Questions

Apa saja format file yang didukung MetaFast?

MetaFast dirancang untuk menangani hampir semua aset digital Anda, termasuk format gambar (JPG, PNG), video (MP4, MOV), dan bahkan file vektor yang rumit seperti Adobe Illustrator (.AI), .EPS, dan .SVG.

Apakah saya butuh kunci API AI sendiri untuk menggunakannya?

Ya. MetaFast berfungsi sebagai alat alur kerja (workflow) yang canggih untuk menghubungkan file Anda dengan penyedia AI pilihan Anda (Gemini, OpenAI, dll). Dengan kunci API Anda sendiri, Anda punya kontrol penuh atas penggunaan dan biaya.

Bisakah saya mengedit hasil dari AI sebelum disimpan?

Tentu! Ini adalah fitur andalan MetaFast versi Lengkap. Anda akan mendapatkan tab "Review & Edit" di mana Anda bisa meninjau dan menyempurnakan setiap judul, deskripsi, dan kata kunci sebelum file akhir disimpan.

Apa perbedaan utama antara versi Lite dan Lengkap?

Versi Lite fokus pada kecepatan dan fungsi esensial: analisis dan penanaman metadata. Versi Lengkap adalah solusi total untuk alur kerja profesional, menambahkan fitur seperti pencari file duplikat, pemantauan folder, inspeksi konten, dan editor interaktif.

Apakah file asli saya diunggah ke server MetaFast?

Tidak pernah. Keamanan Anda adalah prioritas. MetaFast berjalan 100% lokal di komputer Anda. Hanya thumbnail gambar terkompresi dan beresolusi sangat rendah yang dikirim ke penyedia AI untuk dianalisis. File asli Anda tidak pernah meninggalkan komputer Anda.

Apakah MetaFast bisa berjalan di Mac atau Linux?

Untuk saat ini, MetaFast hanya tersedia secara eksklusif untuk Windows 10 dan Windows 11 (64-bit).

Kebijakan Privasi

MetaFast for Arabsstock

1. Data yang Dikumpulkan dan Disimpan

Ekstensi ini hanya memproses informasi yang benar-benar diperlukan untuk berfungsi:

  • Data lokal: API key, preferensi pengguna, status lisensi (disimpan di chrome.storage.local).
  • Data sementara: konten form (judul, tag, kategori) di halaman contributor Arab Stock.
  • Tidak dikumpulkan: identitas pribadi, email, password, data keuangan, lokasi, atau riwayat browsing.

2. Cara Penggunaan Data

  • Mengisi otomatis metadata (judul, tag, kategori).
  • Menyimpan preferensi agar tidak perlu diatur ulang.
  • API key hanya digunakan untuk komunikasi langsung dengan Google Gemini atau OpenAI.

3. Pihak Ketiga

Ekstensi dapat berinteraksi dengan:

  • Google Gemini API (generativelanguage.googleapis.com)
  • OpenAI API (api.openai.com)
  • Google Apps Script (jika lisensi diaktifkan)

Masing-masing memiliki kebijakan privasi sendiri yang disarankan untuk dibaca pengguna.

4. Keamanan Data

  • Data sensitif disimpan lokal, tidak dikirim ke server developer.
  • Kami tidak menjual atau membagikan data pengguna.
  • Disarankan menggunakan API key khusus untuk ekstensi ini.

5. Hak Pengguna

  • Menghapus data dengan menghapus ekstensi atau membersihkan storage browser.
  • Mengganti/mencabut API key kapan saja.
  • Menolak memasukkan data opsional (blacklist kata atau lisensi).

6. Pengungkapan Sesuai Chrome Web Store

Ekstensi ini hanya mengakses konten situs pada halaman contributor Arab Stock. Tidak ada akses ke identitas pribadi, keuangan, kesehatan, autentikasi, komunikasi pribadi, lokasi, histori web, atau aktivitas pengguna di luar situs target.

7. Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini sewaktu-waktu. Perubahan material akan diumumkan melalui halaman publik atau repositori ekstensi.

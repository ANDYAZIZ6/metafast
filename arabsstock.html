Cara Menggunakan

Ikuti 6 langkah mudah ini untuk mulai menggunakan MetaFast for Arabsstock

1

Install Extension

Klik tombol "Add to Chrome" di Chrome Web Store. Ekstensi akan otomatis terinstall di browser Anda dalam hitungan detik. Pastikan Chrome versi 90 atau lebih baru.

2

Aktivasi Lisensi

Masukkan license key yang Anda dapatkan. Aktivasi hanya sekali, tersimpan di cloud, bisa digunakan di perangkat mana saja. Join grup WhatsApp kami untuk mendapatkan license key.

3

Setup API Keys

Masukkan Gemini atau OpenAI API keys di tab Settings. Bisa multiple keys (satu per baris) untuk auto-failover. API keys tersimpan lokal dan tidak pernah dikirim ke server kami.

4

Buka Arabsstock Portal

Pergi ke halaman contributor.arabsstock.com/en/warehouse?type=Images. Side panel ekstensi akan otomatis muncul di sebelah kanan. Pastikan Anda sudah login ke akun Arabsstock Anda.

5

Configure & Start

Pilih processing mode (auto/selected/single), pilih Keyword Language (English/Arabic/Both), pilih Strategi Keyword (single/double/mixed), lalu klik tombol Start Automation. AI akan otomatis fill metadata ke form Arabsstock.

6

Review & Save

Review hasil metadata yang di-generate, edit jika perlu, lalu form akan auto-submit. Checkbox foto akan otomatis di-uncheck setelah selesai. Selesai! Ulangi untuk batch berikutnya.

Cara Install & Menggunakan Ekstensi

üì¶ Tersedia di Chrome Web Store

MetaFast for Adobe Stock sekarang tersedia secara official di Chrome Web Store. Instalasi mudah dengan satu klik, automatic updates, dan keamanan terjamin oleh Google.

Install di Chrome
1

Download & Install

Klik tombol "Add to Chrome" di Chrome Web Store. Ekstensi akan otomatis terinstall di browser Anda dalam hitungan detik.

2

Aktivasi Lisensi

Masukkan license key yang Anda dapatkan. Aktivasi hanya sekali, tersimpan di cloud, bisa digunakan di perangkat mana saja.

3

Setup API Keys

Masukkan Gemini atau OpenAI API keys di tab Settings. Bisa multiple keys (satu per baris) untuk auto-failover.

4

Buka Adobe Stock Portal

Pergi ke halaman contributor.stock.adobe.com/uploads. Side panel ekstensi akan otomatis muncul di sebelah kanan.

5

Pilih Mode & Mulai

Pilih Generate Mode (title/keywords/both), pilih Strategi Keyword, lalu klik tombol proses. AI akan otomatis fill metadata ke form Adobe Stock.

6

Review & Submit

Review hasil metadata yang di-generate, edit jika perlu, lalu klik Save Work di Adobe Stock. Selesai!
